Showcase

Doctor Financial Life System (Api, WebApp and "DevOps")
Made only for doctors, so they can resolve their financial life.
The user fills in its registration and feeds the system with all the procedures, payments, receipts and more and the app keeps control of all the income and expenditure, suggests the best places to work and best clinics to make contract with so the doctor has maximum profit and doesn't have to worry.

I've coded this whole project by myself at work using Vue.js and Node.js (Adonis.js Framework). My manager needed this project to be up and running in a very short period, so I bet that I could do it in a month. This is one of my favorites projects and I'm very proud of it.
HowToTrain Workout App (Api and Website)
A mobile app to help you make the best of your training routine, be it on the gym or by yourself so that you can reach the body and health that you want.

I've made the website for the app and also most of the API wich is written in PHP using laravel.
Event Building Social Network
